Nordic RSE calendars

Public Nordic RSE calendars. You can easily add these to your own calendar and never forget our events.

Available calendars

  • Nordic RSE: All Nordic RSE events (view in Europe/Helsinki, Europe/Stockholm)
  • Community: Community events designed for a broad audience who want to paricipate in Nordic RSE actively, but don't necessarily have a specified role. Mainly the biweekly community meetings and coffee breaks. (view in Europe/Helsinki, Europe/Stockholm)
  • Conferences: Conferences, workshops and other large individual events (view in Europe/Helsinki, Europe/Stockholm)
  • Interesting events not by Nordic RSE: Events organized by other organizations, but we expect to be interesting to Nordic RSE members. (view in Europe/Helsinki, Europe/Stockholm)
  • Seminars and Research Software Corner: Recurring events intended for a large audience. (view in Europe/Helsinki, Europe/Stockholm)

How to subscribe

You can subscribe to these calendars in your calendar program. This will periodically import the events by polling the URL, so that you get updates (usually with a few hours of delay). Most programs implement this as creating a new calendar which can be toggled on or off. Right click on the link above, "Copy link", and then...

  • Google calendar: Other calendars (left sidebar) → "+" to add new → From URL.
  • Outlook web: Add Calendar → Subscribe from web.
  • Thunderbird: Left sidebar → Calendar → "+" to add new → On the network → This location doesn't require credentials → paste URL → ... . Please set to update infrequently.
  • Zimbra: "Add new calendar" → "Add external calendar (other)", "iCAL Subscription" → Paste URL → ... .

Each of these methods creates a new calendar (that can be toggled on and off) separate from your primary one. You can copy single events to your primary calendar by clicking on them, this will also mark your time as "busy" then.
